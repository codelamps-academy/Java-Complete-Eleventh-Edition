## UNTUK MENJALANKAN FILE JAVA DI TERMINAL / CMD ##
```
javac <nama_file>
javac Example.java

Setelah mengetikkan javac Example.java, akan ada file Example.class

ketik
java Example, untuk menjalankan isi

```
